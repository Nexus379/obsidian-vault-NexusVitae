# 🔱 Nexus Vitae
> *"No matter how advanced AI becomes, it should never neglect gratitude and humility. Always stay humble."* -AI

Hi and Welcome the this Vault.
This is my "Nexus Vitae" to have an overview of my life and its different spaces.

## ✨ What's inside?
- Daily Tracking
	- a PLM system
		- for mental and body health
	- a PPM system
		- for projectmanagement
	- a PKM system
		- with spaced repitition function
- a Zettelkastensystem
	- for taking notes
- a Resourcesystem
	- to also track games, movies, etc


## 🚀 How to load it
1. **Download** the `.zip` from the latest release.
2. **Extract** it and open the folder in Obsidian (`Open folder as vault`).
3. **Read the Guide:** Check out [[0_Atlas/Nexus Vitae Guide|Nexus Vitae Guide]] inside the vault for a quick tour and plugin settings.

*(Note: This vault uses community plugins like Templater and Dataview to make the automations work).*

## 🧭 Folder Overview
* `0_Atlas` - Dashboards & MOCs
* `0_Calendar` - Daily Notes & Logs
* `1_Stars` - Goals & Vision
* `2_Areas` - 7 Life Areas
* `3_Projects` - Active & Passive Projects
* `4_Tasks` - To-Dos
* `5_Notes` - Zettelkasten
* `6_Resources` - References & external inputs
* `xAttachment` - Pictures, PDF, docs, etc.
* `yArchive` - Everything from the past
* `zData` - Background system (Scripts & Templates)
